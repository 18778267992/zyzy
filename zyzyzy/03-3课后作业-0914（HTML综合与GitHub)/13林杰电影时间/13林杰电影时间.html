村头王师傅   2018-09-16 17:05

自称是"2016年最值得看的电影"，自称获得过各种国外不知名大奖，我曾在学校被强行组织去观看了点映，只能说，如果本片的导演和谢广坤以及周立波分别被绑在三条铁轨上，我开火车必须选择一条，我会留下谢广坤和周立波的狗命。

村花李建刚   2018-09-11 12:25

本来以为要奔着《丹麦女孩》去了，未曾想后面画风突变。小男主居然能秒懂俩小黑皮的问题出在了哪里，年纪轻轻学识渊博啊。感觉导演比较擅长铺垫和前后呼应，可惜对逻辑严谨性不够重视，因此注定了影片即使能火一下，最终评价也高不到哪儿去。
